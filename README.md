Api sur les recherches d'informations sur les adresses ip



La première fonctionnalité consista à rechercher toutes les informations (dans la limite de l'api) publiques sur une adresse ip.
La deuxième consiste a afficher un point précis sur une map, permettant de localiser approximativement l'adresse ip.
La dernière permet de récupérer l'adresse ip du client et de l'afficher.

J'ai eu des difficultés à afficher une map, j'ai réussi à trouver une solution avec ChatGPT.
